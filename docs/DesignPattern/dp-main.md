# 디자인 패턴(Design Pattern)
{: .no_toc }

수 많은 선배 개발자분들이 소프트웨어 개발 경험을 토대로 정리한 개념이 디자인 패턴입니다.
디자인 패턴은 공통적인 소프트웨어 설계 문제에 대한 해결 방법을 제시하며,
이를 잘 이해하고 적용한다면 좋은 소프트웨어를 개발할 수 있을 것입니다.
